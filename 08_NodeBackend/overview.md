# Tutorial: Build a Node.js + Express Backend with SQLite, CRUD, and JWT Auth

## Goals

- Create a **backend server** using **Node.js** + **Express**.
- Connect it to a **SQLite database**.
- Expose **CRUD operations** using **endpoints**:
    - Get all items
    - Get one item
    - Insert an item
    - Update an item
    - Delete an item
- Support **authentication** with security endpoints using **JWT**:
    - Signup
    - Login
- Include **protected routes** so only authenticated users can call write operations (insert, update, delete).

You will build an API for managing a resource called **Items**, plus a **Users** table for authentication.
