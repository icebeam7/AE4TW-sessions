# CRUD with React

In this tutorial, you will create your first React app. It will fetch and display posts from an API, and allow the user to add, edit, and delete posts.

## Step 1: Download Node.js

Visit the official Node.js download page:

- [https://nodejs.org/en/download/](https://nodejs.org/en/download/)

Download the installer for your operating system using the **"Get a prebuilt Node.js"** option:

- **Windows**: Windows Installer
- **macOS**: macOS Installer
- **Linux**: Standalone Binary
