## Step 10. Edit the employee registration page (Part 2 – Edit an existing employee)

Let’s reuse the previous script in the employee registration page so when an employee is selected from the list, the form fields are pre-filled with the employee details. In registration.php, include the get_employee script right after the initialization of the PHP variables:

```php
include 'get_employee.php';
```
