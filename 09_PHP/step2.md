## Step 2. Set up the project

1.	In your web server’s root directory (www folder), create a new folder with the name Employees. Then, extract the content of the zip file provided in this tutorial (Employees.zip) and paste the files there. 

<img width="2176" height="1060" alt="image" src="https://github.com/user-attachments/assets/76e21b23-68b5-40b8-a229-f36539bc0ebf" />

2.	Open VS Code and open the Employees folder that you created.

3.	Change the extension of each file to .php

4.	Create new PHP files:
*	db_connection.php
*	get_employee.php
*	process_delete.php
*	process_employee.php
