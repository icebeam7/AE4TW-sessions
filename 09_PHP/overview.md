# PHP Tutorial

The goal of this tutorial is to learn how to store information in a MySQL database from a website using PHP.

## Installation

If you use your own computer, you can install **Apache (web server), PHP (server scripting technology), MySQL (database)** altogether by installing **WampServer**:

1. Download from [here](https://wampserver.aviatechno.net/).

2. You can refer to [this guide](https://eecs.csuohio.edu/~sschung/CIS408/StepbyStepGuidetoDownloadInstallWampServerMySQL_2020.pdf) for installing it.

If your OS is Mac or Linux, please install XAMP. A guide for XAMPP is available [here](https://medium.com/smack-thoughts/how-to-install-wamp-and-xampp-8848e24ddba2).

Another way is to use a pre-configured GitHub Codespaces environment. Follow the steps listed [here](https://github.com/dipaish/mysql-phpmyadmin-docker).
